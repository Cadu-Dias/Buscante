# Buscante

Esta é uma aplicação usada para pesquisar livros presentes numa API externa da GOOGLE a qual disponibiliza diversos livros, conforme o assunto

## Ferramentas usadas
  - Angular
  - TypeScript
  - Componentes da RxJS
  - Google API

## Comandos para roda a aplicação
```
1. npm install -g @angular/cli
2. npm install
3. ng serve
```
